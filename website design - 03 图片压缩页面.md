```
┌─────────────────────────────────────────────────────────────┐
│  🛠️ 实用工具网站  [首页] [文件转换] [图片压缩] [图片剪裁] [格式转换] [图片水印] │
└─────────────────────────────────────────────────────────────┘
```

## 功能特点
- 支持批量上传压缩
- 多种压缩级别选择
- 实时预览效果对比
- 显示压缩比例和文件大小
- 保持图片质量的智能压缩
