# 图片水印页面 UI 设计

```
┌─────────────────────────────────────────────────────────────┐
│  🛠️ 实用工具网站  [首页] [文件转换] [图片压缩] [图片剪裁] [格式转换] [图片水印] │
└─────────────────────────────────────────────────────────────┘
```

## 功能特点
- 支持文字和图片水印
- 丰富的字体和颜色选择
- 9个位置可选择放置
- 可调节透明度
- 实时预览水印效果
- 支持批量添加水印
